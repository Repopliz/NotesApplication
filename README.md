Η παρούσα εργασία πραγματεύεται τον σχεδιασμό και την ανάπτυξη μιας εφαρμογής διαχείρισης σημειώσεων για το λειτουργικό σύστημα Android. 
Κύριος στόχος της εργασίας είναι η δημιουργία ενός εύχρηστου και αποδοτικού εργαλείου, που επιτρέπει στον χρήστη να οργανώνει την καθημερινότητά του μέσω της άμεσης καταγραφής πληροφοριών τοπικά στη συσκευή του. 

Η βασική προσέγγιση στηρίζεται στη χρήση της γλώσσας προγραμματισμού Kotlin και του σύγχρονου αρχιτεκτονικού προτύπου MVVM (Model-View-ViewModel).  

Για την αποθήκευση και διαχείριση των δεδομένων χρησιμοποιήθηκε η βιβλιοθήκη Room Persistence Library (SQLite), ενώ για την ασύγχρονη εκτέλεση εργασιών αξιοποιήθηκαν τα Kotlin Coroutines. 
Η διεπαφή χρήστη σχεδιάστηκε με βάση τις αρχές του Material Design, προσφέροντας λειτουργίες CRUD, κατηγοριοποίηση, αναζήτηση σε πραγματικό χρόνο και δυναμικό φιλτράρισμα περιεχομένου.
